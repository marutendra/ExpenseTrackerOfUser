<h3> This is a backend-based Expense Tracker application built using Spring Boot and MongoDB.
The project focuses on managing users and tracking their expenses using REST APIs.
This project helped me understand backend development concepts and MongoDB relationships.</h3>

<h1> Features</h1>

Create, Update, Delete User

Store user details (Name, ID, Password)

Add expenses for a specific User

Fetch all expenses using User ID

Maintain relationship between User and Expense collections

<h1>Technologies Used</h1>

Java

Spring Boot

MongoDB

Spring Data MongoDB

REST APIs
